# JuwelyHunterWorld

## はじめに

- このゲームはUnityを使用した2Dゲームの実装を学ぶために作成されているものです。
- ゲームとして、スタートからゲームクリアまで遊べるように作りました。

## 作品の概要

- ２Ｄスクロールアクションです。  
  マップから各ステージへ入り、最後にボスと戦うゲームになります。

### デモプレイ

#### 操作方法

ステージ

- 移動・・・「←」キー、「→」キー
- ジャンプ・・・Spaceキー
- 攻撃（矢を放つ）・・・Enterキー、Ｆキー、Ｚキー

ワールドマップ

- 移動・・・「↑」キー、「↓」キー、「←」キー、「→」キー
- ステージに入る・・・Enterキー

#### ギミック

| 参考画像 |  ギミック  | 説明                                                                                               |
| :------- | :--------: | :------------------------------------------------------------------------------------------------- |
|          |    攻撃    | 矢を放ちます。画面左上に表示されている残弾が０になると、打てません。                               |
|          |  ダメージ  | プレイヤーが敵や砲弾にあたるとダメージを受けます。プレイヤーのＨＰが０になるとゲームオーバーです。 |
|          |  ステージ  | Enterキーで入場できます。鍵を使用しないと入れない場合も。                                          |
|          | ゴール看板 | ステージクリアの目印です。クリアすると踊りだします。                                               |
|          |     敵     | ハリネズミ。あたると痛いと思われます。                                                             |
|          |    ボス    | 倒すとゲームクリア。攻撃は当然痛い。                                                               |
|          |    大砲    | 砲弾を発射してきます。砲弾にあたると痛い。                                                         |
|          |     針     | 刺さったらゲームオーバー。最初からやり直しです。                                                   |
|          |    宝箱    | アイテムボックス。「矢」「回復」「鍵」のいずれかが入手できます。                                   |
|          |  スイッチ  | 移動床を起動するスイッチです。                                                                     |
|          |   スコア   | ステージに配置されている色つきのアイテムに触れることで取得できます。色によって点数が変わります。   |
